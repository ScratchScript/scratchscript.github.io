Control flow

ScratchScript incorporates control flow structures from both languages. This includes for loops from TypeScript and repeat loops from Scratch.

ScratchScript is also lexically scoped, which means all control flow must be encased in curly brackets ({}).

If/else

if(condition) { // ... } else if(condition2) { // ... } else { // ... }

While

while(condition) { // ... if(breakCondition) { break; } }

Repeat

repeat(times) { // times must be a number // ... if(breakCondition) { break; } }

For

Like in Typescript, for loops have the syntax for(definition; condition; change):

for(let x = 0; x < 100; x++) { // ... if(breakCondition) { break; } }
